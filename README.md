# Audio Cloud

## 作成の背景
- チームでの映像作品制作の経験で感じた、ファイル管理の煩わしさを解消するため制作した。

## 実装の工夫したところ
- 一目でどのようなファイルか判別するために波形イメージの可視化やタグ付けの機能をつけた

## 苦戦したところ
- 波形イメージの表示方法をいろいろ模索して実装した

## これからアップデートできそうなところ
- ファイルをS3などのクラウドストレージにアップロードして実際に使えるようにする
